# Ecommerce System
A fully dynamic functional ecommerce, pet food inventory system created with HTML5_CSS3 & BOOTSTRAP_AJAX_JS_OOP-PHP_MYSQL.

## Features
#### For admin
 - Manage admin
 - Manage cart
 - Manage products
 - Manage pets
 - Issue customer adoption ticket (pending)
#### For customer
 - View products
 - View specific products
 - Buy products
 - Add to cart products
 - Pay products
 - Adopt pet (pending)
